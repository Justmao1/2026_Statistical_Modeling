# 参赛计划书（简化版）

> **大赛名称：** 第十二届全国大学生统计建模大赛
> **大赛主题：** "服务国家战略 创新统计赋能"
> **论文题目：** 《数字经济对区域碳排放强度的影响研究——基于面板回归与机器学习的实证分析》

---

## 一、选题背景与研究意义

### 1.1 现实背景

中国提出"双碳"目标（2030碳达峰、2060碳中和），同时数字经济快速发展，已成为经济增长新引擎。

**核心问题：** 数字经济能否有效降低碳排放？哪些数字经济因素对减碳最重要？

### 1.2 研究意义

- **理论意义**：探讨数字经济和碳排放之间的关系，丰富相关文献
- **方法意义**：将传统回归分析与机器学习相结合，用SHAP进行可解释性分析
- **实践意义**：为政策制定提供数据支撑

### 1.3 契合大赛主题

- **"服务国家战略"**：对接"双碳"目标 + 数字中国战略
- **"创新统计赋能"**：统计回归 + 机器学习预测 + SHAP可解释性

### 1.4 创新点

1. **方法结合**：将面板回归（分析因果）和机器学习（预测+SHAP解释）相结合
2. **指标构建**：用熵权法构建数字经济综合指数（4维度、12指标）
3. **政策参考**：基于SHAP特征重要性排序，提出针对性政策建议

## 三、研究方法

### 3.1 总体思路

```
数据采集与清洗
     ↓
构建数字经济综合指数（熵权法）
     ↓
  ┌──────────┬──────────┐
  ↓                     ↓
统计回归分析         机器学习预测
（因果推断）         （预测+解释）
  │                     │
  ↓                     ↓
① 固定效应回归      ① 随机森林
② 中介效应分析      ② XGBoost
③ 稳健性检验        ③ SHAP可解释分析
  │                     │
  └──────────┬──────────┘
             ↓
     结论与政策建议
```

### 3.2 统计回归部分

#### 模型一：基准回归（双向固定效应）

$$CO2_{it} = \alpha_0 + \alpha_1 DE_{it} + \gamma X_{it} + \mu_i + \delta_t + \varepsilon_{it}$$

- $CO2_{it}$：碳排放强度（CO2/GDP）
- $DE_{it}$：数字经济综合指数
- $X_{it}$：控制变量
- $\mu_i$：省份固定效应，$\delta_t$：时间固定效应

#### 模型二：中介效应（三步法）

```
路径一: 数字经济 → 产业结构优化 → 碳排放强度↓
路径二: 数字经济 → 绿色技术创新 → 碳排放强度↓
```

使用逐步回归法（Baron & Kenny三步法）+ Sobel检验

### 3.3 机器学习部分

#### 模型选择

| 模型 | 特点 | 调参方法 |
|------|------|---------|
| 随机森林 | 稳健、不易过拟合 | 5折交叉验证 |
| XGBoost | 预测精度高 | 5折交叉验证 + GridSearch |

#### 评价指标

| 指标 | 说明 |
|------|------|
| R² | 解释方差比例，越大越好 |
| RMSE | 均方根误差，越小越好 |
| MAE | 平均绝对误差，越小越好 |

#### SHAP可解释性分析

用SHAP值分析每个特征对碳排放预测的贡献方向和大小。

**需要做3张SHAP图：**

| 图表 | 作用 |
|------|------|
| SHAP蜂群图 | 展示所有特征的重要性排序 |
| SHAP柱状图 | 直观对比各因素影响大小 |
| SHAP依赖图 | 展示某个特征的非线性影响 |

### 3.4 稳健性检验（4项）

| 序号 | 方法 | 操作 |
|------|------|------|
| 1 | 替换因变量 | 用人均碳排放替代碳排放强度 |
| 2 | 替换自变量构建方法 | 用PCA替代熵权法构建指数 |
| 3 | 缩尾处理 | 连续变量1%和99%缩尾 |
| 4 | 排除直辖市 | 剔除北京、天津、上海、重庆 |

### 7.2 核心代码参考

```python
import pandas as pd
import numpy as np
from sklearn.ensemble import RandomForestRegressor
from xgboost import XGBRegressor
from sklearn.model_selection import GridSearchCV
from sklearn.metrics import r2_score, mean_squared_error, mean_absolute_error
import shap
import matplotlib.pyplot as plt

# 1. 读取数据
df = pd.read_csv('panel_data.csv')

features = [
    '互联网普及率', '移动电话普及率', '光缆长度',
    '电信业务总量', 'IT服务收入', '软件业务收入',
    '金融覆盖广度', '金融使用深度', '金融数字化程度',
    '数字专利数', 'RD人员密度', '技术市场成交额',
    'ln人均GDP', '产业结构', '城镇化率', '能源结构'
]
target = '碳排放强度'

# 2. 划分训练集/测试集
train = df[df['年份'] <= 2020]
test  = df[df['年份'] >= 2021]
X_train, y_train = train[features], train[target]
X_test,  y_test  = test[features],  test[target]

# 3. 训练模型
xgb = XGBRegressor(n_estimators=200, max_depth=5, learning_rate=0.05, random_state=42)
xgb.fit(X_train, y_train)

rf = RandomForestRegressor(n_estimators=200, random_state=42)
rf.fit(X_train, y_train)

# 4. 模型对比
for name, model in [('XGBoost', xgb), ('RandomForest', rf)]:
    y_pred = model.predict(X_test)
    print(f"{name}: R²={r2_score(y_test, y_pred):.4f}, "
          f"RMSE={np.sqrt(mean_squared_error(y_test, y_pred)):.4f}")

# 5. SHAP分析
explainer = shap.TreeExplainer(xgb)
shap_values = explainer.shap_values(X_test)

shap.summary_plot(shap_values, X_test, feature_names=features)
shap.summary_plot(shap_values, X_test, feature_names=features, plot_type='bar')
```

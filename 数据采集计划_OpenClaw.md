## 批次②：经济与社会基础数据（国家统计局）

### 需要的指标

| 指标 | 变量名 | 说明 | 用途 |
|------|--------|------|------|
| 地区生产总值(GDP) | GDP | 亿元 | 计算碳排放强度、人均GDP |
| 人均地区生产总值 | PGDP | 元 | 控制变量（取ln） |
| 第二产业增加值 | GDP_2 | 亿元 | 产业结构 = 二产/GDP |
| 第三产业增加值 | GDP_3 | 亿元 | 产业结构高级化 = 三产/二产 |
| 年末常住人口 & 城镇人口 | POP, URB_POP | 万人 | 城镇化率 = 城镇/总 |

### 数据源

- **网站**: 国家统计局·国家数据
- **网址**: https://data.stats.gov.cn/
- **路径**: 首页 → 地区数据 → 分省年度数据

### OpenClaw 提示词

```
任务：从国家统计局网站批量采集省级经济社会数据

请访问 https://data.stats.gov.cn/ ，执行以下操作：

1. 在页面顶部选择"地区数据"→"分省年度数据"
2. 分别查询以下5组指标，时间范围选择2011-2023年，地区选择全部省份：

   第1组 - 地区生产总值(GDP)：
   - 路径：国民经济核算 → 地区生产总值
   - 需要：各省各年的GDP（亿元）

   第2组 - 人均GDP：
   - 路径：国民经济核算 → 人均地区生产总值
   - 需要：各省各年的人均GDP（元）

   第3组 - 产业增加值：
   - 路径：国民经济核算 → 地区生产总值
   - 需要：各省各年的第二产业增加值、第三产业增加值（亿元）

   第4组 - 人口数据：
   - 路径：人口 → 年末常住人口/城镇人口
   - 需要：各省各年的年末常住人口数、城镇常住人口数（万人）

3. 每组数据查询后，点击下载或导出Excel
4. 将所有数据合并整理为一张CSV表

输出CSV格式，包含列：
省份, 年份, GDP(亿元), 人均GDP(元), 第二产业增加值(亿元), 第三产业增加值(亿元), 常住人口(万人), 城镇人口(万人)

文件命名为：nbs_economic_data.csv

注意：
- 排除港澳台和西藏
- 如某年数据缺失，标记为空值
- 省份名称统一使用简称（如"北京"而非"北京市"）
```

## 📊 数据处理计划

### 采集完成后的变量计算

| 最终变量 | 计算公式 | 所需原始数据 |
|---------|---------|------------|
| 碳排放强度 (CI) | CO2排放量 / GDP | 批次①②  |
| 人均碳排放 | CO2排放量 / 常住人口 | 批次①②  |
| 人均GDP (lnPGDP) | ln(人均GDP) | 批次② |
| 产业结构 (IS) | 第二产业增加值 / GDP | 批次② |
| 城镇化率 (URB) | 城镇人口 / 常住人口 | 批次② |
| 能源结构 (ES) | 煤炭消费量 / 能源消费总量 | 批次⑦ |
| 互联网普及率 | 互联网用户 / 常住人口 × 100 | 批次②③ |
| 移动电话普及率 | 移动电话用户 / 常住人口 × 100 | 批次②③ |
| 产业结构高级化 (ISU) | 第三产业增加值 / 第二产业增加值 | 批次②（中介变量） |
| 绿色技术创新 (GTI) | ln(绿色专利申请数 + 1) | 批次⑥（中介变量） |

# 计算衍生变量
df['碳排放强度'] = df['CO2排放量'] / df['GDP']
df['人均碳排放'] = df['CO2排放量'] / df['常住人口']
df['ln人均GDP'] = np.log(df['人均GDP'])
df['产业结构'] = df['第二产业增加值'] / df['GDP']
df['城镇化率'] = df['城镇人口'] / df['常住人口']
df['能源结构'] = df['煤炭消费量'] / df['能源消费总量']
df['互联网普及率'] = df['互联网用户数'] / df['常住人口'] * 100
df['移动电话普及率'] = df['移动电话用户数'] / df['常住人口'] * 100
df['产业结构高级化'] = df['第三产业增加值'] / df['第二产业增加值']
df['绿色技术创新'] = np.log(df['绿色专利申请数'] + 1)

df.to_csv('panel_data.csv', index=False)

1. **省份名称统一**：所有数据统一使用省份简称（如"北京"而非"北京市"），合并前务必检查

5. **数据来源引用**：记录每个指标的具体来源，论文中需要注明
